A personalized, interactive 3D space experience built with **Three.js** and **Vite**.

## How to Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Features

-   **Interactive Star Field**: Click stars to reveal personalized messages.
-   **Planetary Exploration**: Visit planets and land on the Moon.
-   **Proximity Interactions**: Walk up to objects to trigger events.
-   **3D Finale**: A special heart-warming ending.

## Customization

-   **Music**: Place your audio file at `public/music.mp3`.
-   **Messages**: Edit the `LOVE_REASONS` array in `main.js`.
